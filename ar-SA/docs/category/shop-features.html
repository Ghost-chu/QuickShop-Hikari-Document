Shop Features

All features related to a quickshop.

📄️ Admin Shop (Unlimited Shop)

QuickShop provides a feature that allow you set a shop to unlimited space (for buying shop) or unlimited stock (for selling shop).

📄️ Shop Benefits

Shop Benefits feature allow you split your shop reward to multiple players.

📄️ Blacklist System

Server administrator can disable shop creation at/for world or items.

📄️ Display Item

QuickShop provide a cool feature that let player identify shops item.

📄️ Per shop permission management

QuickShop-Hikari provide a system to allow control every single shop's permissions.

📄️ Price Limiter

QuickShop provide a price limiter allow you specific the item's min or max price on your server.

📄️ Protection Checker

QuickShop use a util called Protection Checker to prevent players create shops at their had no build access positions (e.g Spawn).

📄️ Shop Protection

Player Protection

📄️ Ranks

QuickShop provides a ranks system to allow control the maximum shops that player can own.

📄️ Batch remove shops

If you have a massive amounts of the shop need to remove, there are some ways you can do that quickly.

📄️ Create and purchase

This page of document will help you learn how to create and purchase a quickshop.

📄️ Shop Blocks

Not only Chests, but also all Containers all supported used for QuickShop containers!

📄️ Shop Fees

QuickShop allow server administrator setup the shops fees for creating, changing prices.

📄️ Shop Purger

Shop Purger allow server administrator purge the shops if the player in idle or banned.

📄️ Shop Search

QuickShop have a built-in shop searcher will allow you search the shops on your server without load any chunks.

📄️ Stacking Shop

QuickShop allow you sell/buy multiple item in bundles.
